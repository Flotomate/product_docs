Flotomate RDP Middleware Installation Guide¶

Describes Installation Steps for Flotomate RDP Middleware – 1.0.0

System Requirement:¶

Product built needs to be installed and configured in the Main Server.

Downloading RDP Middleware¶

You can download the RDP Middleware from the below link:

https://s3.ap-south-1.amazonaws.com/flotomate-customer-releases/latest/rdp+middleware/rdp_middleware

Manual Installation Setup:¶

After downloading the RDP Middleware, follow the below steps to install RDP on to your server

  1. Log in to the terminal of the Main Server.
  2. Change directory to the RDP Installation file.
  1. Check whether you have the permission to execute the file.

  1. Make Sure you have the execute permission to build the file. If it does not have the execute permission then you can change it by using the following command:

    sudo chmod 777 rdp_middleware

  2. Run the rdp_middleware executable file.

  1. Close terminal after installation finishes.
  1. Now you have to enable RDP in the product. Log in to the product’s Technician portal.
  2. Go to Admin (A Navigation tab) >> RDP Configuration (Asset Management).
  1. In the RDP Configuration dialog box, enable RDP.
  2. Click on Configure RDP.
  1. Now the product is configured with RDP.
