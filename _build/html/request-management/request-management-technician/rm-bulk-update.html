Bulk UpdateÂ¶

The product allows Technicians to change Request information of multiple Requests at once.

To perform Bulk Update:

  • Go to the Request List View.

  • Select more than one Request.

  • The Bulk Update button appears above the list area. Click on Bulk Update, and a dialog box opens.

    You can change the following things:

    1. Change the assignee of the Requests.
    1. Modify the Technician Group.
    2. Change four classifiers (Status, Priority, Urgency, and Impact).
    3. Add new tags or append to the existing ones.
    4. Change Category, Source, Department, and Location.

  • When you are done hit Update, and your changes are saved.
