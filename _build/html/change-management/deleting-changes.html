Deleting ChangesÂ¶

Deleting is a bulk operation which means you can delete multiple Changes at once.

  • Go to Change >> List View.
  • Select a Change or Changes from the list area.
  • Click Archive, and you get a confirmation message; click Yes to delete the selected Change/Changes.
