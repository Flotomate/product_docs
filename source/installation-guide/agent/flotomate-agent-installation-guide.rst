**********************************
Flotomate Agent Installation Guide
**********************************

Describes Installation Steps for Flotomate Windows & Linux Agent.

System Requirement :
====================

**Target Machine OS**: 

- Windows 7 and above.
- Linux.

**OS Architecture**: 

- Windows (32 & 64 bit).
- Linux (64bit).

.. note:: Currently we have only tested the Linux Agent in Ubuntu 16.04.
